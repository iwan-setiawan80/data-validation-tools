# Validation Rules

## Field Validation

- Required fields cannot be empty.
- Text fields must not exceed the maximum length.
- Numeric values must be within the allowed range.
- Dates must follow the YYYY-MM-DD format.

## Duplicate Check

- Record IDs must be unique.
- Email addresses should not be duplicated.

## Consistency Check

- Country and city must match.
- Status values must use predefined options.

## Business Rules

- Birth date cannot be in the future.
- End date must be later than the start date.
- Required documents must be provided before approval.
